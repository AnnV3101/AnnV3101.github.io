Как написать калькулятор на Python за 10 минут

📅 15 февраля 2026 ⏱️ 7 мин чтения

Сегодня мы напишем простой калькулятор на Python, который умеет складывать, вычитать, умножать и делить. Это отличная практика для новичков — вы научитесь работать с функциями, условиями и вводом данных.

Шаг 1. Простая версия

Начнём с самого простого калькулятора, который выполняет одну операцию за раз:

a = float(input("Введите первое число: "))
b = float(input("Введите второе число: "))
op = input("Введите операцию (+, -, *, /): ")

if op == "+":
  print("Результат:", a + b)
elif op == "-":
  print("Результат:", a - b)
elif op == "*":
  print("Результат:", a * b)
elif op == "/":
  if b != 0:
    print("Результат:", a / b)
  else:
    print("Ошибка: деление на ноль!")
else:
  print("Неизвестная операция")

Шаг 2. Добавляем цикл

Чтобы калькулятор работал постоянно, пока пользователь не захочет выйти, обернём код в цикл while:

while True:
  a = float(input("Первое число: "))
  b = float(input("Второе число: "))
  op = input("Операция (+, -, *, /) или 'q' для выхода: ")
  
  if op == 'q':
    break
  
  # здесь тот же код с условиями
  print() # пустая строка для разделения

Шаг 3. Оформляем в функции

Сделаем код более структурированным, вынеся операции в отдельные функции:

def add(a, b):
  return a + b

def subtract(a, b):
  return a - b

def multiply(a, b):
  return a * b

def divide(a, b):
  if b == 0:
    return "Ошибка: деление на ноль"
  return a / b

operations = {
  "+": add,
  "-": subtract,
  "*": multiply,
  "/": divide
}

while True:
  a = float(input("a = "))
  b = float(input("b = "))
  op = input("операция: ")
  if op in operations:
    print("=", operations[op](a, b))
  else:
    print("Неизвестная операция")

Шаг 4. Улучшаем интерфейс

Можно добавить поддержку ввода выражений одной строкой, например "2 + 2":

while True:
  expr = input("Введите выражение (или 'q'): ")
  if expr == 'q':
    break
  try:
    a, op, b = expr.split()
    a = float(a)
    b = float(b)
    if op == "+":
      print("=", a + b)
    # ... остальные операции
  except:
    print("Ошибка ввода")

Заключение

Теперь у вас есть собственный калькулятор! Попробуйте улучшить его: добавить поддержку более сложных выражений, возведение в степень, работу с несколькими числами или графический интерфейс.
