Что такое HTML и CSS простыми словами

📅 20 февраля 2026 ⏱️ 4 мин чтения

HTML и CSS — это основа любого сайта. Если сравнить сайт с домом, то HTML — это стены, комнаты и фундамент, а CSS — это дизайн интерьера, цвет стен и расстановка мебели.

HTML — скелет страницы

HTML (HyperText Markup Language) — это язык разметки. Он состоит из тегов, которые описывают структуру документа. Например:

<h1>Заголовок страницы</h1>
<p>Это абзац текста.</p>
<img src="photo.jpg" alt="Фото">

Браузер читает эти теги и показывает пользователю страницу с заголовком, текстом и картинкой.

CSS — внешность

CSS (Cascading Style Sheets) отвечает за то, как элементы выглядят: цвета, размеры, шрифты, отступы, анимации.

h1 {
  color: red;
  font-size: 24px;
}
p {
  color: gray;
  line-height: 1.5;
}

Как они работают вместе

HTML создаёт элементы, а CSS их стилизует. Например, вот так выглядит простая карточка:

<div class="card">
  <h2>Привет!</h2>
  <p>Я карточка</p>
</div>
.card {
  border: 1px solid black;
  padding: 20px;
  border-radius: 10px;
  background: #f0f0f0;
}

Где писать код

HTML и CSS можно писать даже в Блокноте, но удобнее использовать специальные редакторы: VS Code, Sublime Text, Notepad++. Главное — сохранять файлы с расширениями .html и .css.

Краткий итог

16:35

HTML — это структура, CSS — это стиль. Без HTML у страницы нет содержимого, без CSS она выглядит как чёрно-белый текст 90-х. Вместе они творят магию современного веба.
