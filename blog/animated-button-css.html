Создаём анимированную кнопку на CSS

📅 10 февраля 2026 ⏱️ 5 мин чтения

Кнопки — один из самых важных элементов интерфейса. Анимированная кнопка привлекает внимание и делает сайт современным. Сегодня сделаем красивую неоновую кнопку в стиле киберпанк.

Базовая структура HTML

<button class="neon-btn">Наведи на меня</button>

Базовые стили

Начнём с простого оформления:

.neon-btn {
  background: transparent;
  border: 2px solid #00ffff;
  color: #00ffff;
  padding: 15px 30px;
  font-size: 18px;
  border-radius: 50px;
  cursor: pointer;
  transition: 0.3s;
}

Добавляем свечение

Эффект неона достигается с помощью box-shadow и text-shadow:

.neon-btn {
  box-shadow: 0 0 10px #00ffff;
  text-shadow: 0 0 5px #00ffff;
}

.neon-btn:hover {
  background: #00ffff;
  color: #0a0c0f;
  box-shadow: 0 0 30px #00ffff;
}

Пульсация

Добавим анимацию пульсации при наведении:

@keyframes pulse {
  0% { box-shadow: 0 0 10px #00ffff; }
  50% { box-shadow: 0 0 40px #00ffff; }
  100% { box-shadow: 0 0 10px #00ffff; }
}

.neon-btn:hover {
  animation: pulse 1.5s infinite;
}

Два цвета (Magenta + Cyan)

Сделаем кнопку с двумя неоновыми цветами:

.duo-btn {
  background: transparent;
  border: 2px solid #ff00ff;
  color: #ff00ff;
  padding: 15px 30px;
  border-radius: 50px;
  position: relative;
  overflow: hidden;
  transition: 0.3s;
  z-index: 1;
}

.duo-btn::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: linear-gradient(90deg, #00ffff, #ff00ff);
  transition: 0.4s;
  z-index: -1;
}

.duo-btn:hover {
  color: #0a0c0f;
  border-color: transparent;
}

.duo-btn:hover::before {
  left: 0;
}

Заключение
