Словари (dict)

Словарь — это неупорядоченная коллекция пар "ключ-значение". Ключи должны быть уникальными и неизменяемыми.

person = {
    "name": "Анна",
    "age": 25,
    "city": "Москва"
}

# Альтернативный способ
person2 = dict(name="Иван", age=30, city="Казань")

Доступ к элементам

person = {"name": "Анна", "age": 25}

print(person["name"]) # Анна
print(person.get("age")) # 25
print(person.get("phone", "Не указан")) # Не указан

Добавление и изменение

person = {"name": "Анна"}

# Добавление
person["age"] = 25

# Изменение
person["name"] = "Мария"

Удаление

person = {"name": "Анна", "age": 25, "city": "Москва"}

del person["city"] # удалить по ключу
age = person.pop("age") # удалить и вернуть значение
person.clear() # очистить весь словарь

Методы словарей

person = {"name": "Анна", "age": 25}

print(person.keys()) # все ключи
print(person.values()) # все значения
print(person.items()) # все пары (ключ, значение)

for key, value in person.items():
    print(f"{key}: {value}")

Множества (set)

Множество — это неупорядоченная коллекция уникальных элементов.

fruits = {"яблоко", "банан", "апельсин", "яблоко"}
print(fruits) # {'апельсин', 'банан', 'яблоко'} (дубликат удалён)

# Из списка во множество
numbers = [1, 2, 2, 3, 3, 3, 4]
unique = set(numbers) # {1, 2, 3, 4}

Операции с множествами

a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b) # объединение: {1, 2, 3, 4, 5, 6}
print(a & b) # пересечение: {3, 4}
print(a - b) # разность: {1, 2}
print(a ^ b) # симметрическая разность: {1, 2, 5, 6}

Методы множеств

fruits = {"яблоко", "банан"}

fruits.add("апельсин") # добавить элемент
fruits.remove("банан") # удалить элемент (ошибка, если нет)
fruits.discard("груша") # удалить, если есть (без ошибки)
fruits.clear() # очистить множество
