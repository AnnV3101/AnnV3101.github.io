Урок 5: Циклы while и for

⏱️ 10 минут

Цикл while

Цикл while выполняет блок кода, пока условие истинно.

count = 1
while count <= 5:
    print("Счёт:", count)
    count += 1

Бесконечный цикл и break

Можно создать бесконечный цикл и прервать его по условию с помощью break.

while True:
    cmd = input("Введи 'stop' для выхода: ")
    if cmd == 'stop':
        break
    print("Продолжаем...")

Цикл for

Цикл for используется для перебора элементов последовательности (список, строка, диапазон).

for i in range(5): # 0,1,2,3,4
    print(i)

fruits = ["яблоко", "банан", "груша"]
for fruit in fruits:
    print(fruit)

range() с шагом

for i in range(1, 10, 2): # нечётные числа от 1 до 9
    print(i)

continue

Оператор continue прерывает текущую итерацию и переходит к следующей.

for i in range(1, 6):
    if i == 3:
        continue
    print(i)

Практическое задание

Напишите программу, которая выводит таблицу умножения на 5 (от 1 до 10), используя цикл for. Затем перепишите её с циклом while.
