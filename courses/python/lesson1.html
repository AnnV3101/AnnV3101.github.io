Урок 1: Установка Python и VS Code

⏱️ 6 минут

Что нужно сделать

Чтобы начать программировать на Python, нужно установить интерпретатор и среду разработки. Мы будем использовать Python 3 и редактор VS Code (он бесплатный и удобный).

Установка Python

1. Перейди на официальный сайт python.org.
2. Скачай последнюю версию Python для своей ОС (Windows, macOS, Linux).
3. Запусти установщик и обязательно отметь галочку "Add Python to PATH" (это важно!).
4. Дождись завершения установки.

Проверка установки

Открой командную строку (терминал) и введи:

python --version

Если ты видишь что-то вроде Python 3.12.0 — всё ок.

Установка VS Code

1. Скачай VS Code с code.visualstudio.com.
2. Установи, оставляя все настройки по умолчанию.
3. Запусти VS Code, установи расширение для Python (нажми на иконку расширений слева, найди "Python" от Microsoft и установи).

Первая программа

Создай новый файл с расширением .py, например hello.py, и напиши:

print("Привет, мир!")

Нажми кнопку запуска (треугольник) в правом верхнем углу — внизу должно появиться "Привет, мир!". Поздравляю, ты написал первую программу! 🎉
