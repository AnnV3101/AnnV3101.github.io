Урок 1: Установка Python и VS Code

⏱️ 6 мин

Что нужно сделать

1. Скачай Python с официального сайта python.org.

2. При установке обязательно отметь галочку "Add Python to PATH".

3. Скачай VS Code с code.visualstudio.com.

4. Установи расширение Python в VS Code.

5. Проверь: открой терминал и напиши python --version.
