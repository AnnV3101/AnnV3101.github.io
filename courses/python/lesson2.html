Урок 2: Переменные и типы данных

⏱️ 5 минут

Что такое переменная?

Переменная — это именованная область памяти, в которой хранятся данные. В Python переменная создаётся простым присваиванием значения.

Основные типы данных

  • int — целые числа (например, 5, -3, 1000)
  • float — числа с плавающей точкой (3.14, -0.5, 2.0)
  • str — строки (текст), заключаются в кавычки: "Привет", 'Python'
  • bool — логические значения: True или False

Примеры кода

# Целое число
age = 25
print(age)

# Число с плавающей точкой
price = 99.90
print(price)

# Строка
name = "Анна"
print("Меня зовут", name)

# Логический тип
is_student = True
print(is_student)

Как узнать тип переменной

В Python есть встроенная функция type(), которая возвращает тип данных:

x = 42
print(type(x)) # <class 'int'>

y = "Hello"
print(type(y)) # <class 'str'>

Динамическая типизация

В Python тип переменной определяется автоматически и может меняться:

var = 10 # int
var = "теперь строка" # теперь str
print(var)

Практическое задание

Создай переменные разных типов и выведи их на экран. Попробуй сложить строку и число — посмотри, что получится (и почему это ошибка).
