Урок 2: Типы данных и приведение типов

⏱️ 6 минут

Основные типы данных в JavaScript

В JS есть 8 основных типов данных. Самые часто используемые:

  • number – числа (целые и дробные)
  • string – строки (текст)
  • boolean – логические значения (true/false)
  • undefined – значение не присвоено
  • null – "ничего", пустое значение
  • object – объекты (массивы, функции, объекты)

Примеры

let age = 25; // number
let name = "Анна"; // string
let isStudent = true; // boolean
let city; // undefined
let data = null; // null
let user = {name: "Анна", age: 25}; // object
let colors = ["красный", "синий"]; // object (массив)

Оператор typeof

Позволяет узнать тип переменной:

console.log(typeof 42); // "number"
console.log(typeof "Привет"); // "string"
console.log(typeof true); // "boolean"
console.log(typeof undefined); // "undefined"
console.log(typeof null); // "object" (историческая особенность)
console.log(typeof {}); // "object"

Приведение типов (преобразование)

JavaScript часто сам преобразует типы, что может приводить к неожиданностям.

Строки и числа

console.log("5" + 3); // "53" (конкатенация)
console.log("5" - 3); // 2 (числовое вычитание)
console.log("5" * "2"); // 10 (умножение)

Явное преобразование

let str = "123";
let num = Number(str); // 123
let float = parseFloat("3.14"); // 3.14
let int = parseInt("10.5"); // 10

let value = 42;
let strValue = String(value); // "42"
let boolValue = Boolean(0); // false (0, "", null, undefined, NaN -> false)

Практическое задание

Напиши программу, которая запрашивает у пользователя два числа через prompt (они приходят как строки), складывает их и выводит результат. Подсказка: не забудь преобразовать в числа.
