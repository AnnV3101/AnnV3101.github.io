Урок 3: Условные операторы if/else

⏱️ 7 минут

Зачем нужны условия?

Условия позволяют программе принимать решения в зависимости от данных. Например, проверять возраст, сравнивать числа, реагировать на действия пользователя.

Конструкция if

Выполняет код, только если условие истинно (true).

let age = 18;
if (age >= 18) {
  console.log("Ты совершеннолетний");
}

if...else

Если условие ложно, выполняется блок else.

let age = 16;
if (age >= 18) {
  console.log("Ты совершеннолетний");
} else {
  console.log("Ты ещё несовершеннолетний");
}

if...else if...else

Для нескольких условий используется else if.

let mark = 85;
if (mark >= 90) {
  console.log("Отлично");
} else if (mark >= 75) {
  console.log("Хорошо");
} else if (mark >= 60) {
  console.log("Удовлетворительно");
} else {
  console.log("Плохо");
}

Логические операторы

Можно комбинировать условия:

  • && – И (оба условия истинны)
  • || – ИЛИ (хотя бы одно истинно)
  • ! – НЕ (отрицание)
let age = 25;
let hasLicense = true;
if (age >= 18 && hasLicense) {
  console.log("Можно водить машину");
}

let isWeekend = true;
let isHoliday = false;
if (isWeekend || isHoliday) {
  console.log("Можно отдыхать");
}

Тернарный оператор

Краткая запись if...else для простых случаев:

let age = 20;
let status = age >= 18 ? "взрослый" : "ребёнок";
console.log(status); // "взрослый"

Практическое задание

Напиши программу, которая запрашивает у пользователя число и выводит:

  • "Положительное", если число больше 0
  • "Отрицательное", если меньше 0
  • "Ноль", если равно 0
