Урок 1: Переменные и вывод в консоль

⏱️ 5 минут

Что такое JavaScript?

JavaScript — это язык программирования, который делает веб-страницы интерактивными. Он работает прямо в браузере.

Переменные

Переменные используются для хранения данных. В JavaScript есть три способа объявить переменную: let, const и (устаревший) var.

let name = "Анна"; // строковая переменная
const age = 25; // константа (нельзя изменить)
let isStudent = true; // логическое значение
let price; // объявление без значения (undefined)

let — переменная может меняться. const — значение нельзя переназначить.

Вывод в консоль

Для отладки удобно использовать console.log(). Это выводит информацию в консоль браузера (F12 → Console).

console.log("Привет, мир!");
console.log(name); // выведет значение переменной name
console.log("Мне", age, "лет");

Простейшая программа

Создайте файл script.js и подключите его к HTML. Либо пишите код прямо в консоли браузера для тренировки.

let userName = prompt("Как тебя зовут?");
console.log("Привет, " + userName);
alert("Добро пожаловать, " + userName);

Практическое задание

Объявите три переменные с вашими именем, возрастом и городом. Выведите их в консоль в виде: "Меня зовут [имя], мне [возраст] лет, я из [город]".
