Свойства flex-элементов

В прошлом уроке мы изучили свойства flex-контейнера. Теперь разберём, как управлять отдельными элементами.

order

Изменяет порядок элементов. По умолчанию все элементы имеют order: 0. Чем меньше значение, тем раньше элемент.

.item1 { order: 2; }
.item2 { order: 1; } /* будет первым */
.item3 { order: 3; }

flex-grow

Определяет, как элемент будет растягиваться, если есть свободное место. По умолчанию 0 (не растягивается).

.item { flex-grow: 1; } /* элемент будет занимать всё свободное место */

Пример с flex-grow: второй элемент растягивается

1
2 (растёт)
3

flex-shrink

Определяет, как элемент будет сжиматься, если не хватает места. По умолчанию 1.

flex-basis

Задаёт начальный размер элемента до распределения свободного пространства.

.item { flex-basis: 200px; }

flex (сокращение)

Объединяет flex-grow, flex-shrink и flex-basis:

.item { flex: 1 0 200px; } /* grow: 1, shrink: 0, basis: 200px */

align-self

Переопределяет align-items для конкретного элемента.

.item { align-self: flex-end; }

Пример с align-self: второй элемент прижат к низу

Практическое задание

Создай галерею из 6 карточек, которые:

  • Располагаются в ряд с переносом
  • Имеют минимальную ширину 200px
  • Растягиваются, если есть место
  • Имеют одинаковую высоту
  • Третья карточка должна быть сдвинута вниз (используй align-self)
