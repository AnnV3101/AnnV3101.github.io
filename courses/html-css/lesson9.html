Что такое Grid?

CSS Grid — это двумерная система компоновки, которая позволяет размещать элементы в строках и столбцах одновременно. В отличие от Flexbox (одномерный), Grid даёт полный контроль над сеткой.

Основные понятия

  • grid-контейнер — элемент с display: grid
  • grid-элементы — прямые потомки контейнера
  • строки (rows) — горизонтальные линии
  • столбцы (columns) — вертикальные линии

display: grid

Чтобы создать grid-контейнер:

.container {
  display: grid;
}

grid-template-columns и grid-template-rows

Задают количество и размер столбцов и строк:

.container {
  display: grid;
  grid-template-columns: 100px 200px 100px;
  grid-template-rows: 100px 150px;
}

Единица fr

fr — это доля свободного пространства. Аналог flex-grow в Flexbox.

.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
}

Сетка с долями: 1fr, 2fr, 1fr

1fr
2fr
1fr

gap

Задаёт отступы между ячейками:

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

repeat()

Упрощает повторяющиеся значения:

.container {
  grid-template-columns: repeat(3, 1fr);
  /* то же самое что: 1fr 1fr 1fr */
}

Практическое задание

Создай сетку 3x3 с размерами:

  • Первый столбец — 200px
  • Второй столбец — 1fr
  • Третий столбец — 1fr
  • Все строки по 100px
  • Отступы между ячейками — 10px
