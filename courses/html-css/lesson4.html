Урок 4: Ссылки и изображения

⏱️ 8 минут

Ссылки (тег a)

Тег <a> создаёт гиперссылку на другую страницу, файл или якорь. Основной атрибут — href, который указывает адрес.

<a href="https://example.com">Перейти на сайт</a>
<a href="/about/">О нас</a> (внутренняя ссылка)
<a href="#section1">Перейти к разделу 1</a> (якорь)

Открытие в новой вкладке

Атрибут target="_blank" открывает ссылку в новой вкладке:

<a href="https://example.com" target="_blank">Открыть в новой вкладке</a>

Ссылка-картинка

Можно сделать картинку ссылкой, обернув тег <img> в <a>:

<a href="https://example.com">
  <img src="logo.png" alt="Логотип">
</a>

Изображения (тег img)

Тег <img> вставляет изображение. Это одиночный тег (не требует закрытия). Основные атрибуты:

  • src — путь к файлу изображения
  • alt — альтернативный текст (если изображение не загрузилось)
  • width и height — размеры (можно задавать в пикселях или процентах)
<img src="cat.jpg" alt="Милый котик" width="300">

Поддерживаемые форматы

  • JPEG — для фотографий, много цветов
  • PNG — для графики с прозрачностью
  • GIF — для анимаций
  • SVG — векторные изображения (масштабируются без потери качества)
  • WebP — современный формат, лучше сжимает

Пути к изображениям

Можно использовать абсолютные или относительные пути:

<!-- Абсолютный путь (с сайта) -->
<img src="https://example.com/images/photo.jpg">

<!-- Относительный путь (файл в той же папке) -->
<img src="photo.jpg">

<!-- В папке на уровень выше -->
<img src="../images/photo.jpg">

figure и figcaption

Теги <figure> и <figcaption> используются для группировки изображения с подписью:

<figure>
  <img src="photo.jpg" alt="Описание">
  <figcaption>Красивый закат в горах</figcaption>
</figure>

Практическое задание

Создай HTML-страницу, на которой будут:

  • Ссылка на любой сайт, открывающаяся в новой вкладке
  • Изображение (можно использовать ссылку на картинку из интернета)
  • Изображение-ссылка — клик по картинке ведёт на другой сайт
  • Фигура с подписью (используй figure и figcaption)
