Урок 3: Списки (ul, ol)

⏱️ 6 минут

Маркированные списки (ul)

Тег <ul> создаёт маркированный список. Каждый элемент списка обозначается тегом <li>.

<ul>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ul>

В браузере это выглядит как:

  • Первый пункт
  • Второй пункт
  • Третий пункт

Нумерованные списки (ol)

Тег <ol> создаёт нумерованный список. Нумерация может быть арабскими цифрами, римскими, буквами — это настраивается CSS.

<ol>
  <li>Первый шаг</li>
  <li>Второй шаг</li>
  <li>Третий шаг</li>
</ol>

Результат:

  1. Первый шаг
  2. Второй шаг
  3. Третий шаг

Вложенные списки

Списки можно вкладывать друг в друга, создавая многоуровневую структуру.

<ul>
  <li>Фрукты</li>
  <ul>
    <li>Яблоки</li>
    <li>Бананы</li>
  </ul>
  <li>Овощи</li>
</ul>

Атрибуты для ol

У тега <ol> есть полезные атрибуты:

  • type — тип нумерации: 1 (числа), A (заглавные буквы), a (строчные), I (римские), i (строчные римские).
  • start — начальное значение (например, start="5").
<ol type="A" start="3">
  <li>Третий пункт</li>
  <li>Четвёртый пункт</li>
</ol>

Практическое задание

Создай страницу с рецептом, используя маркированный список для ингредиентов и нумерованный — для шагов приготовления.
