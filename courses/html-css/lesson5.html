Урок 5: Таблицы

⏱️ 7 минут

Зачем нужны таблицы?

Таблицы используются для представления структурированных данных: расписания, прайс-листы, сравнительные характеристики и т.д.

Основные теги таблиц

  • <table> — контейнер для всей таблицы
  • <tr> — строка таблицы (table row)
  • <td> — ячейка с данными (table data)
  • <th> — ячейка-заголовок (table header) — текст жирный и по центру

Простая таблица

<table border="1">
  <tr>
    <th>Имя</th>
    <th>Возраст</th>
  </tr>
  <tr>
    <td>Анна</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Иван</td>
    <td>30</td>
  </tr>
</table>

Результат:

Имя Возраст
Анна 25
Иван 30

Объединение ячеек

  • colspan — объединяет ячейки по горизонтали (сколько столбцов)
  • rowspan — объединяет ячейки по вертикали (сколько строк)
<table border="1">
  <tr>
    <th colspan="2">Общий заголовок</th>
  </tr>
  <tr>
    <td>Ячейка 1</td>
    <td>Ячейка 2</td>
  </tr>
</table>

Структурные теги

Для больших таблиц используют логические блоки:

  • <thead> — шапка таблицы
  • <tbody> — тело таблицы (основные данные)
  • <tfoot> — подвал таблицы (итоги, суммы)
<table>
  <thead>
    <tr><th>Товар</th><th>Цена</th></tr>
  </thead>
  <tbody>
    <tr><td>Ноутбук</td><td>50000</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Итого</td><td>50000</td></tr>
  </tfoot>
</table>

Стилизация таблиц

С помощью CSS можно сделать таблицы красивыми:

table {
  width: 100%;
  border-collapse: collapse; /* убирает двойные границы */
}
th, td {
  border: 1px solid cyan;
  padding: 0.5rem;
  text-align: left;
}
th {
  background: rgba(0, 255, 255, 0.2);
}
tr:hover {
  background: rgba(255, 0, 255, 0.1);
}

Практическое задание

Создай таблицу расписания уроков:

  • Колонки: День недели, Время, Предмет
  • Добавь 3-4 строки с данными
  • Используй thead и tbody
  • Добавь CSS-стили для красоты
