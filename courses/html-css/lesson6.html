Урок 6: Формы и элементы ввода

⏱️ 9 минут

Что такое формы?

Формы используются для сбора данных от пользователей: регистрация, вход, обратная связь, поиск и т.д. Основной тег — <form>.

Основные атрибуты формы

  • action — URL, куда отправляются данные
  • method — метод отправки: get (в URL) или post (в теле запроса)
<form action="/submit" method="post">
  ... элементы формы ...
</form>

Элемент input

Самый универсальный элемент. Тип задаётся атрибутом type:

  • text — текстовое поле
  • password — поле для пароля (символы скрыты)
  • email — поле для email (проверка формата)
  • number — поле для чисел
  • checkbox — флажок (можно выбрать несколько)
  • radio — переключатель (только один из группы)
  • file — загрузка файла
  • submit — кнопка отправки
  • reset — кнопка сброса
<input type="text" name="username" placeholder="Введите имя">
<input type="password" name="password">
<input type="email" name="email">
<input type="number" name="age" min="1" max="100">

Текстовая область (textarea)

Для многострочного текста:

<textarea name="comment" rows="5" cols="40"></textarea>

Выпадающий список (select)

<select name="city">
  <option value="msk">Москва</option>
  <option value="spb">Санкт-Петербург</option>
  <option value="kazan">Казань</option>
</select>

Метки (label)

Тег <label> связывает текст с полем ввода. Удобно и для доступности:

<label for="username">Имя пользователя:</label>
<input type="text" id="username" name="username">

Группировка полей (fieldset и legend)

<fieldset>
  <legend>Контактные данные</legend>
  <label>Email: <input type="email" name="email"></label>
  <label>Телефон: <input type="tel" name="phone"></label>
</fieldset>

Пример формы

Валидация

HTML5 предоставляет встроенную валидацию:

  • required — обязательное поле
  • minlength / maxlength — минимальная/максимальная длина
  • min / max — для чисел и дат
  • pattern — проверка по регулярному выражению
<input type="text" required minlength="3" pattern="[A-Za-z]+">

Практическое задание

Создай форму регистрации со следующими полями:

  • Имя (обязательное, минимум 2 символа)
  • Email (обязательное, проверка формата)
  • Пароль (обязательное, минимум 6 символов, тип password)
  • Пол (radio)
  • Страна (select с 3-4 вариантами)
  • Согласие на обработку данных (checkbox, обязательное)
  • Кнопка "Зарегистрироваться"
