Что такое формы?

Формы используются для сбора данных от пользователей: регистрация, вход, обратная связь, поиск и т.д. Основной тег — <form>.

Основные атрибуты формы

  • action — URL, куда отправляются данные
  • method — метод отправки: get (в URL) или post (в теле запроса)
<form action="/submit" method="post">
  ... элементы формы ...
</form>

Элемент input

Самый универсальный элемент. Тип задаётся атрибутом type:

  • text — текстовое поле
  • password — поле для пароля (символы скрыты)
  • email — поле для email (проверка формата)
  • number — поле для чисел
  • checkbox — флажок (можно выбрать несколько)
  • radio — переключатель (только один из группы)
  • file — загрузка файла
  • submit — кнопка отправки
  • reset — кнопка сброса
<input type="text" name="username" placeholder="Введите имя">
<input type="password" name="password">
<input type="email" name="email">
<input type="number" name="age" min="1" max="100">

Текстовая область (textarea)

<textarea name="comment" rows="5" cols="40"></textarea>

Выпадающий список (select)

<select name="city">
  <option value="msk">Москва</option>
  <option value="spb">Санкт-Петербург</option>
  <option value="kazan">Казань</option>
</select>

Метки (label)

<label for="username">Имя пользователя:</label>
<input type="text" id="username" name="username">
